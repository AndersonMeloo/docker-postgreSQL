#  PostgreSQL + Docker

---

## 🚀 Tecnologias utilizadas 

### 🗄️ Banco de Dados
- PostgreSQL (via Docker)

### 🐳 Infraestrutura
- Docker  
- Docker Compose

### 1️⃣ Clonar o repositório
```bash
git clone https://github.com/SEU-USUARIO/fullstack-docker-postgres.git
```
